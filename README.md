# Curriculum Vitae Creation Form

This project is a web-based form for creating a Curriculum Vitae (CV). Users can input their personal information, including name, gender, birth date, email, phone number, address, summary, experience, education, skills, and languages. Trimestral Activity 1 Unity 1 of 'Lenguajes de Marcas'. 'Grado Superior DAW, Universidad Europea.'

## Features

- **Profile Picture**: Upload a profile picture.
- **Personal Information**: Input fields for name, last name, and gender.
- **Birth Date**: Date input with validation to ensure the date is not today or in the future.
- **Contact Information**: Input fields for email and phone number with validation.
- **Address**: Input field for address.
- **Summary**: Text area for a brief summary about the user.
- **Experience**: Text area for listing experience.
- **Education**: Text area for listing education history.
- **Skills**: Text area for listing skills.
- **Languages**: Text area for listing languages and proficiency levels.
- **On Submit**: Redirection to confirmation page.

## Validations

- **Name and Last Name**: Required text inputs.
- **Gender**: Required radio buttons.
- **Birth Date**: Required date input with a maximum date set to yesterday.
- **Email**: Required email input with a pattern to ensure a valid email format.
- **Phone Number**: Required phone number input with a pattern to ensure a valid format and an `inputmode` to restrict input to numeric characters.
- **Address**: Required text input.
- **Summary, Experience, Education, Skills, and Languages**: Text areas with maximum length restrictions.
